# ProjetoReavP3
Esse projeto trata-se do projeto de reavaliação da matéria "Projeto de Software".

O sistema escolhido para reproduzir foi o de Produtividade Acadêmica.

Os Padrões de projeto utilizados foram:

#Singleton:

Foi utilizado para manter apenas uma intância de dois objetos, o primeiro é o objeto "DataBase", que possui a função de armazenar os dados dos colaboradores e dos projetos. O outro objeto que utiliza esse padrão é o objeto "Entry", que é responsável por ler todas as entradas necessárias para o funcionamento do sistem.

#MVC:

Todo o sistema foi desenvolvido a partir da estrutura "MVC", logo, todos as classes se encaixam em seus respectivos "Packages", sendo eles "Models", "View", "Control".
